Microsoft Financial Statement Analysis

For this analysis, we were provided with the dataset in the form of excel spreadsheet, containing different financial statement reports of Microsoft.

The follwing steps were performed in order to arrive at these dashboard reports.
      * Understanding the data and deciding on which information is key to be presented to the business users
      * Data cleaning and preprocessing to bring the data into a desired format suitable for visualization
      * Designing the pages for the dashboard and the layout for each page by choosing the right visual
      * Explored the Inforiver tool, which is one of the fastest way to build financial reports in Power BI
