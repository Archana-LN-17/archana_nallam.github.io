Friends series analysis Onyx data

This was introduced by Onyx data as part of their DataDNA challenge on LinkedIn, which included dataset about the famous "Friends" series. Friends is an American television sitcom created by David Crane and Marta Kauffman, which aired on NBC from September 22, 1994, to May 6, 2004, lasting ten seasons. The data included details like the number of episodes for each season, directors for each season and the ratings provided.
